# Hashing Methods Demonstration

## Project Overview
This C++ implementation demonstrates two fundamental hashing concepts:
- Division Method for key hashing
- Quadratic Probing for collision resolution
The system provides step-by-step trace outputs showing insertion, collision handling, and lookup processes.
